# bmp-line-drawer

Простой инструмент на C++ для чтения 24-битных BMP-изображений, отображения их в ASCII-виде в консоли и рисования X-образных линий поверх изображения. После изменения картинка сохраняется обратно в формате BMP.

## Возможности

- Чтение BMP-файла (24-bit)
- Парсинг заголовков BMP (BITMAPFILEHEADER, BITMAPINFOHEADER)
- Вывод изображения в консоль (`#` — чёрный пиксель, `.` — белый)
- Отрисовка диагональных линий (X) поверх изображения
- Сохранение результата в новый BMP-файл
